那是最**美好**的时代，那是最**糟糕**的时代；  
那是**智慧**的年头，那是**愚昧**的年头；  
那是**信仰**的时期，那是**怀疑**的时期；  
那是**光明**的季节，那是**黑暗**的季节；  
那是**希望**的春天，那是**失望**的冬天；  
我们面前**无所不有**，我们面前**一无所有**；  
我们大家都在**直升天堂**，我们大家都在**直下地狱**。

简而言之，那个时代和当今这个时代是如此相似，因而一些吵嚷不休的权威们也坚持认为，不管它是好是坏，都只能用 “最……” 来评价它。

-- 《双城记》 人民文学出版社，石永礼、赵文娟译
